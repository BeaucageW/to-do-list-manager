# CLI To-Do list manager

## Instructions
Just run the program and you will be ready to start making a list. Anything you type in the terminal that isn't recognized as a command will simply be added to the list. For commands description / list, see below.

## Commands

* "quit" / "exit" : Exits the program.

* "print" : Displays the current list.
